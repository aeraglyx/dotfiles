# Aeraglyx's dotfiles

My personal dotfiles repository.


## Installation

Requires git and stow.

```bash
git clone git@github.com:aeraglyx/dotfiles
cd ~/dotfiles
stow *
```


## Theme

Uses a custom theme called Onyx. I made these stand-alone ports, the rest is currently in this repo and I keep tweaking it.

- [Blender](https://github.com/aeraglyx/onyx)
- [Neovim](https://github.com/aeraglyx/onyx.nvim)
